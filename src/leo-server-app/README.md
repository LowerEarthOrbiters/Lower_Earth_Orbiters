Server Application

## Getting Started

First, build and transpile the code:

```bash
npm -i
npm run build
```

Then, start the following application using the below command:

```bash
npm run start:dev
```

To start the client and server app concurrently, use:

```bash
npm run all
```

By default, the server will run on 3001 port.

To run tests:

```bash
npm run test
```
